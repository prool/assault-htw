# README #

This is the MUD Assault: High Tech War as run by Demortes. This is the latest version as run at assault.demortes.com port 6660.
Not actively maintained, kept up here for nostalgia reasons and tinkering.

* git clone it and run "make" from the src directory.

* cd ../area

* ../src/ack 4000
